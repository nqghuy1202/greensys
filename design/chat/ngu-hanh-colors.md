# Bảng màu Ngũ Hành

| Hành | Key | `--c-main` | Màu |
|---|---|---|---|
| Kim (Metal) | `kim` | `#2563EB` | Blue-600 |
| Moc (Wood) | `moc` | `#16A34A` | Green-600 |
| Thuy (Water) | `thuy` | `#B45309` | Amber-700 |
| Hoa (Fire) | `hoa` | `#DC2626` | Red-600 |
| Tho (Earth) | `tho` | `#7B4F24` | Cognac Brown |

## Biến CSS đầy đủ theo từng theme

| Biến | Mô tả |
|---|---|
| `--c-main` | Màu chính (accent) |
| `--c-hover` | Màu hover (đậm hơn main) |
| `--c-tint` | Màu nền nhạt (tint background) |
| `--c-border` | Màu viền |
| `--c-medium` | Màu trung gian |
| `--c-dark` | Màu đậm |
| `--c-focus` | Màu focus ring (rgba) |
